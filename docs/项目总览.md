## 1. 项目定位

精准匹配是一个基于 Web 的相亲匹配平台，本质是：
> 给一个人贴很多标签 -> 按特定标签筛选 -> 相当于线上 Excel 表格

核心功能：
- 发布信息（贴标签）
- 按标签筛选搜索
- 管理后台查看用户列表
- 举报功能（看评语决定怎么处理）
- 保证金机制（确保双方见面信息正确）

不需要的功能：
- 站内消息/聊天（不是社交软件）
- 支付系统（只是保证金凭证上传）

## 4. 启动方式

方式 1：一键启动
```
双击运行 E:\website\start-all.bat
```

方式 2：手动启动
```bash
# 后端（终端1）
cd E:\website\2.my-two-website\backend
npm start

# 前端（终端2）
cd E:\website\2.my-two-website\app
npm run dev
```

## 6. 环境变量

后端（`backend/.env`）
```env
PORT=8080
FRONTEND_URL=http://localhost:4000
ADMIN_PASSWORD=your_secure_password
SERVER_ORIGIN=http://your_server_ip
```

前端（`app/.env.development`）
```env
VITE_BASE_URL=/jzxr/
VITE_API_BASE_URL=/jzxr/api
```

生产环境必须检查 `app/.env.production` 中 `VITE_BASE_URL` 设置。

## 9. 核心功能摘要

| 功能 | 说明 |
|------|------|
| 用户注册/登录 | 手机号 + 密码，bcrypt 哈希，Token 认证 |
| 资料发布 | 40+ 字段的用户资料表单 |
| 搜索筛选 | 关键词搜索 + 多条件字段筛选 |
| 多条件匹配 | AND 逻辑匹配，显示满足全部条件的结果 |
| 评价体系 | 用户对用户评价，计算平均分 |
| 举报系统 | 举报 + 管理后台审核处理 |
| 保证金 | 上传支付凭证，管理员审核 |
| 联系方式解锁 | 付费解锁，AES-256-CBC 加密存储 |
| 管理后台 | 用户管理、审核、统计、日志 |
| 滑动验证码 | 人机验证 |

## 10. 数据库概览

数据库文件：`data/match.db`（SQLite）

| 表名 | 用途 |
|------|------|
| `auth_users` | 用户认证（手机号、密码哈希） |
| `users` | 用户资料（40+ 字段） |
| `user_ratings` | 用户评价 |
| `user_reports` | 用户举报 |
| `contact_unlocks` | 联系方式解锁记录 |
| `deposits` | 保证金记录 |
| `payment_orders` | 支付订单 |
| `admin_settings` | 管理设置 |
| `admin_logs` | 管理员操作日志 |
